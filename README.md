resistor-label-builder
======================

Resistor Label Builder generates organizational labels for your resistors. Each label contains the resistance and the appropriate 5 band color code. Alternate specifications (Number of rows, columns, subdivision of labels, margins, etc.) can be supplied to use other label sheets. Requires Pillow for Python -- https://pypi.python.org/pypi/Pillow/, Recommended that you use Avery 8593 White File-Folder Labels, as this is what it's designed for.

In short, it'll turn this mass of resistance values: [1, 2.2, 4.7, 5.6, 7.5, 8.2, 10, 15, 22, 27, 33, 39, 47, 56, 68, 75, 82, 100, 120, 150, 180, 220, 270, 330, 390, 470, 510, 680, 820, 1000, 1500, 2200, 3300, 3900, 4700, 5600, 6800, 7500, 8200, 10000, 15000, 22000, 33000, 39000, 47000, 56000, 68000, 75000, 82000,  10000, 150000, 180000, 220000, 330000, 470000,  560000, 680000, 1000000, 1500000, 2000000, 3300000, 4700000, 5600000, 10000000]
